# Contributing to HORUS ROS 2 Infrastructure

We welcome contributions to the HORUS project!

Please see our main [Contribution Guidelines](https://github.com/RICE-unige/horus_sdk/blob/main/CONTRIBUTING.md) in the central SDK repository.

## Reporting Issues

If you find a bug specific to the ROS 2 packages, please open an issue in this repository.
For general SDK issues or feature requests, please use the [Main Repository Issues](https://github.com/RICE-unige/horus_sdk/issues).
